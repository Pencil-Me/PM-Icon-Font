Das Einrichten des PM-Iconset´s ist extrem einfach. Nur eine Zeile auf der Website und schon ist es nutzbar!

Basis: Basis-Installation

Das PM-Iconset kann einfach mit nur einer Zeile HTML auf der Webseite installiert werden. Man braucht nicht mal etwas herunterzuladen oder zu installieren!

  1. Fügen den folgenden Code in den -Bereich der Webseite ein. <link href="//www.webiconfont.com/latest/css/pm-iconset.min.css" rel="stylesheet">
  2. Klopfe dir selbst auf die Schulter für frei-scalierbare-Vektor-Icons-auf-der-eigenen-Webseite mit nur einer Zeile Code.
  3. Schauen Dir die Beispiele an und starte mit dem PM-Iconset!

Erweitert: selbst Hosten

Benutze diese Methode um das PM-Iconset selbst per CSS zu bearbeiten.

  1. Kopiere das PM-Iconset-Verzeichnis in den Webprojekt.
  2. Im -Bereich deines HTML-Textes binde das pm-iconset.min.css ein. <link rel="stylesheet" href="path/to/pm-iconset/css/pm-iconset.min.css">
  3. Schauen Dir die Beispiele an und starte mit dem PM-Iconset!
