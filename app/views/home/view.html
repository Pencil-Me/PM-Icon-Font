PM-Iconset stellt frei skalierbare Vektor-Icons zur Verfügung, die vollständig angepasst werden können - Größe, Farbe, Schatten und vieles mehr... alles was mit CSS beeinflusst werden kann.

  • Ein Font, 200+ Icons

    Das PM-Iconset bietet mehr als 200 Pictogrammen in einer einzigen Font-Sammlung.

  • Volle Kontrolle mit CSS

    Egal ob Farbe, Größe, Schatten oder sogar Animation: Die volle Kontrolle dank CSS.

  • Unendliche Skalierbarkeit

    Das PM-Iconset besteht aus Vektorgrafiken. Das bedeutet, das die Icons immer gestochen scharf sind, egal in welcher Größe.

  • Frei Nutzung

    PM-Iconset ist frei nutzbar für auch kommerzielle Zwecke, ausser der direkte Weiterverkauf.

  • Desktopfreundlich

    Um das PM-Iconset auf dem Desktop zu verwenden, einfach die Referenzliste für Sonderzeichen nutzen.

  • Screen-Reader-kompatibel

    Das PM-Iconset ist, entgegen anderer Iconsets, barrierefrei konzipiert. Ein Leseprogramm wird daher keinen Buchstabensalat ausgeben.
