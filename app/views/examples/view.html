Sobald alles installiert wurde und läuft kann das PM-Iconset in jeden tag eingefügt werden.

Basisnutzung

Füde einfach ein PM-Iconset Icon in ein tag, entweder als CSS-class oder als data-icon.

Icon "workcross" Aufruf via data-icon
Icon "workcross" Aufruf via class
<span data-icon="workcross"></span> Icon "workcross" Aufruf via data-icon<br> <span class="pmiconfont-workcross"></span> Icon "workcross" Aufruf via class
Alle Icons werden via CSS :before aufgerufen.

Spinning Icons

Benutze die pm-spin class um ein rotierendes Icon zu bekommen.

CSS3 animations aren't supported in IE8 - IE9.

Rotate & Flipp

Um Icons zu drehen oder zu spiegeln nutze einfach die pm-rotate-* und pm-flip-* Klassen.

normal
pm-rotate-90
pm-rotate-180
pm-rotate-270
pm-flip-horizontal
pm-flip-vertical

Custom CSS

Alles was man mit CSS font styles machen kann, geht auch mit PM-Iconset.
